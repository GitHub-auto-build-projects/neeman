WebViewController

public class WebViewController: UIViewController, NeemanUIDelegate, NeemanNavigationDelegate

WebViewController displays the contents of a URL and pushes a new instance of itself once the user clicks on a link that causes an URL change. It also provides support for authentication. It makes injecting Javascript into your webapp easy.
